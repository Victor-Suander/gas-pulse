# Gas Pulse

Aplicação web em Python e Flask para processar dados do case técnico Bridge & Co.

## Objetivo

Construir uma aplicação que consolida vendas de postos de combustível, emails de gerentes, preços de referência e gera relatórios.

## Tecnologias

- Python
- Flask
- pandas
- requests
- beautifulsoup4

## Como executar

1. Crie e ative um ambiente virtual Python.
2. Instale as dependências:

```bash
pip install -r requirements.txt
```

3. Inicie a aplicação:

```bash
python run.py
```

4. Acesse `http://localhost:5000` no navegador.
