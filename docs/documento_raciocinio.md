# Documento de Raciocínio

Este documento registra a abordagem inicial para o case técnico Bridge & Co.

- Objetivo: criar uma aplicação web limpa e funcional em Python/Flask.
- Escopo inicial: estrutura de projeto, rota inicial e página home.
- Futuras etapas: implementar processamento de vendas, emails, preços de referência e relatórios.
- Motivação: manter a arquitetura simples, fácil de explicar e escalável.
